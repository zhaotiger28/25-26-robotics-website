# Hornet Robotics Website

Official website for FTC Team 9045 - Hornet Robotics from Greenhill School, Dallas TX.

## Pages

- **index.html** - Home page with hero, about, features, and team preview
- **team.html** - Full team roster with interactive bio modals
- **community.html** - STEM outreach and community initiatives
- **contact.html** - Contact form and social links

## Structure

```
├── index.html
├── team.html
├── community.html
├── contact.html
├── style.css
├── scrambleanimation.js
├── images/
│   ├── hero.JPG
│   ├── placeholder.svg
│   └── teamheadshots/
└── bios/
    └── [membername]bio.txt
```

## Features

- Responsive design with mobile navigation
- Interactive team member modals with auto-loaded bios
- CSS variables for consistent theming (green/gold)
- Auto-detection of team headshot images

## Development

Open `index.html` in a browser or use a local server:

```bash
npx serve .
```
